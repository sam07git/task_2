CSS Fundamentals

Cascading Style Sheets (CSS) are the magic behind the beautiful and stylish web pages we encounter daily. CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a web document written in HTML. It's responsible for how your web content looks and feels, including layout, colors, fonts, and more. CSS separates content (HTML) from its presentation, allowing you to control the appearance of your web pages with ease. SS offers a wide range of properties and values to style elements.
Here are a few commonly used ones:
1.color: Sets the text color (e.g., color: red;).
2.font-size: Adjusts the font size (e.g., font-size: 18px;).
3.margin: Specifies the margin around an element (e.g., margin: 10px;).
4.padding: Sets the padding inside an element (e.g., padding: 5px;).
5.background-color: Defines the background color (e.g., background-color: #f0f0f0;).

JavaScript v/s Node.js


Purpose

JavaScript: JavaScript is a programming language primarily used for client-side web development. It runs in web browsers and is responsible for making web pages interactive and dynamic.
Node.js: Node.js, on the other hand, is a runtime environment for executing JavaScript code on the server-side. It allows JavaScript to be used for server-side programming, handling tasks like file manipulation, networking, and creating web servers.


APIs and Modules

JavaScript: In the browser environment, JavaScript has access to the Document Object Model (DOM) and Browser APIs, which allow it to interact with web pages and perform browser-related tasks.
Node.js: Node.js provides its own set of APIs and modules that are not available in the browser environment. These include modules for file I/O, networking, and building web servers
